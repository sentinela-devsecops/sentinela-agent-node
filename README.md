# sentinela-agent-node
Módulo de varredura distribuído desenvolvido em Node.js 18+. Utiliza Worker Threads para análise de sistemas de arquivos em tempo real com baixo impacto de CPU (&lt; 5%). Realiza a detecção de segredos via Regex, cifragem assimétrica de payloads e resiliência com Exponential Backoff.
